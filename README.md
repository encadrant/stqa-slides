# Slides for the STQA course.

These are an early draft of the slides for the STQA course. They're still a
_work in progress_, and I'll strive to improve them. If you spot any mistake
(even a typo) please let me know.

## Content

- [Black Box Testing (1)](blackbox.pdf): Equivalence partitionning, BVA
- [Black Box Testing (2)](blackbox-suite.pdf): Combinatorial testing, decision
tables, Cause-Effect graphs.
- [White Box Testing (1)](whitebox.pdf): CFGs, all nodes, all decisions,
multiple conditions, MC/DC.
- [White Box Testing (2)](whitebox-suite.pdf): DFGs, all-defs, all-uses, all-du.

## Using these slides

These slides are meant for the 4 ING students (2025-2026). A lot more has been
discussed during the actual course.
